# 🏢 서울 아파트 일반공급 청약 경쟁률 예측 모델 고도화 프로젝트

> **공공데이터 API 기반의 분양 정보를 세션 단위로 재구성하여 서울 아파트 일반공급 경쟁률을 사전 예측하고, SHAP 분석을 통해 청약 수요 결정 요소를 정량적으로 규명한 머신러닝 프로젝트입니다.**

## 📅 프로젝트 기간 및 참여도
* **기간**: 2025.11 ~ 2025.12
* **구성**: 개인 프로젝트 (Data Analyst 1명, CTO 피드백 및 리드)
* **주요 역할**: 데이터 수집/정제, 데이터셋 재구성, 변수 체계 확립, LightGBM 모델링 및 SHAP 기반 해석 (기여도 100%)
* **사용 언어 및 라이브러리**: `Python`, `LightGBM`, `SHAP`, `Pandas`, `NumPy`, `Scikit-learn`

---

## 🎯 1. 프로젝트 목적 및 배경
* **비즈니스 목적**: 사이트 이용자가 입력한 분양 정보를 기반으로 **일반공급 청약 경쟁률을 사전에 예측**해주는 기능을 구현 및 고도화하여, 사용자의 분양 의사결정을 지원함.
* **분석 목표**: 파편화된 공공데이터를 단지·주택형 기준으로 통합하고, 저공급 및 강남 3구 중심의 초고경쟁률 단지 특성을 반영할 수 있는 머신러닝 예측 구조를 설계함.

---

## 🛠 2. 데이터 파이프라인 및 변수 체계 확립

### 📊 데이터 수집 및 재구성
* **데이터 규모**: 공공데이터 API를 기반으로 **2020.02 ~ 2025.11 기간의 데이터 총 1,129건** 수집 및 정제.
* **데이터셋 구조화**: 분양 공고별 단지 및 주택형 정보를 기준으로 파편화된 공공데이터를 세션 단위로 재구성하여, 청약 경쟁률 예측 모델 학습에 최적화된 서울 아파트 데이터셋을 자체 구축함.

### 🧠 피처 엔지니어링 (Feature Engineering)
청약 시장의 주요 의사결정 요소를 반영하기 위해 아래와 같이 변수 체계를 확립했습니다.
* **공급 요인**: 공급세대수, 특별공급/일반공급 구성 비율
* **입지 및 브랜드**: 시군구 정보, 브랜드 영향력, 지역 프리미엄 지표
* **환경 요인**: 분양 및 정책 시행 시점

---

## 📈 3. 모델링 및 데이터 분석 결과

### 🤖 모델 설계 및 타깃 정의
* **타깃 변수(Target)**: `일반공급 접수건수 / 일반공급 공급세대수` (일반공급 경쟁률)
* **핵심 알고리즘**: 범주형 변수 처리가 용이하고 대량의 파생변수 학습이 빠른 **`LightGBM Regressor`**를 중심으로 예측 구조 설계.

### 🔍 탐색적 데이터 분석 (EDA) 및 수요 집중 현상 확인
* **경쟁률 분포 분석**: 데이터 시각화 및 분포 확인 결과, 소수의 초고경쟁 단지가 분포의 극단적인 우측 꼬리(Long Tail)를 형성하는 현상 확인.
* **인사이트**: **저공급 세대수 단지** 및 **강남 3구 프리미엄 브랜드 단지**에 청약 수요가 극도로 집중되는 시장의 불균형성을 정량적으로 확인.

### 📊 SHAP 기반 피처 중요도 분석
SHAP(Shapley Additive exPlanations) 라이브러리를 활용해 모델의 예측 기여도를 분석한 결과, 경쟁률에 미치는 영향력의 순위는 다음과 같았습니다.
1. **공급세대수** (가장 지배적인 영향력, 음의 상관관계)
2. **시군구 (지역 프리미엄)**
3. **브랜드 영향력**

## ⚠️ 4. 프로젝트의 한계점 및 향후 고도화 방향 (CTO 피드백 반영)

### 📌 현 모델의 한계점
* **예측 오차 확대**: 현재 구축된 데이터셋에는 청약 당시의 '실제 시세 변화', '대출 환경(금리, 규제)', '과거 청약 이력' 등 **수요의 강도를 직접적으로 설명하는 외부 변수가 부재**함.
* 이로 인해, 수요가 비정상적으로 몰리는 **초고경쟁 단지 구간(Extreme Values)에서 예측 오차가 확대**되는 한계를 발견함.

### 🚀 향후 모델 개선 및 고도화 계획
CTO 피드백을 바탕으로 모델의 일반화 성능과 편향을 개선하기 위해 아래의 고도화 작업을 진행할 예정입니다.
1. **설명 변수 확장**: 최근 1~2년 시군구 평균 청약 경쟁률/당첨 가점, 미달률 등의 '청약 이력 지표'와 기준금리 변화, LTV/DSR 규제 강도 등의 '거시 경제 환경 변수' 추가 적재.
2. **다양한 모델 앙상블 및 비교**:
   * **`CatBoost`**: 범주형 변수(시군구, 브랜드 등) 처리의 안정성 확보.
   * **`XGBoost`**: 하이퍼파라미터 튜닝을 통한 극단값 예측 완화.
   * **`GLM (일반화 선형 모델)`**: 선형적 성격을 결합하여 모델의 편향(Bias) 개선.
